# Auto-Eject

This is a simple client mod that automatically drops unwanted items to help keeping your inventory cleaner. You can customize the list of unwanted items that will be ejected in the settings page accessible via mod menu.    
By default it auto ejects the following items:
- `rotten_flesh`
- `poisonous_potato`
- `pufferfish`
- `egg`
- `blue_egg`
- `diorite`
- `andesite`
- `granite`
- `warped_roots`
- `nether_sprouts`
- `crimson_roots`
- `oxeye_daisy`
- `poppy`
- `dandelion`
- `azure_bluet`
- `leaf_litter`
- `lily_pad`
- `blue_egg`
> Remember you can configure them
# Features
- Killswitch with Hotkey:   

	Lets you turn Auto Eject on or off instantly without opening the config screen.
- Customizable Eject List:  

	You choose which items are ejected.

- Skip Hotbar:

	Optional toggle to dont auto-eject itmes on the hotbar (on by defaul).

# Config
Configurations are available through the mod menu.
